# v1.3.0

## 关于 / About

一个用于按行处理文本的 VSCode 扩展。A VSCode extension for line-by-line text processing.

### 功能 / Features

| 功能 Feature | 说明 Description |
|---|---|
| **Add Single Quotes** | 为每行添加单引号 `'text'` / Wrap each line with single quotes |
| **Add Double Quotes** | 为每行添加双引号 `"text"` / Wrap each line with double quotes |
| **Add Trailing Comma** | 每行末尾添加逗号（最后一非空行除外）/ Add comma at end of each line (except last non-empty) |
| **Join with Commas** | 多行合并为一行，逗号分隔 / Join all lines with commas into one line |
| **Split by Commas** | 逗号分隔文本拆分为多行 / Split comma-separated text into multiple lines |

### 使用方法 / Usage

1. 打开任意文本文件 / Open any text file
2. **右键点击编辑器 → 文本行处理器** / **Right-click in editor → Text Line Processor**
3. 从弹出菜单中选择操作 / Choose an operation from the QuickPick panel

> 没有选中文本则处理整个文件；选中文本则只处理选中行。
> If no text is selected, the entire file is processed. If text is selected, only selected lines are processed.
